## 문제 설명

무한 수열 A는 다음과 같다.

- A0 = 1

- Ai = A⌊i/P⌋ + A⌊i/Q⌋ (i ≥ 1)

N, P와 Q가 주어질 때, AN을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 3개의 정수 N, P, Q가 주어진다.

## 출력

첫째 줄에 AN을 출력한다.

## 제한

- 0 ≤ N ≤ 1012

- 2 ≤ P, Q ≤ 109

### 예제 입력1 & 예제 출력1

```text
7 2 3

```

```text
7
```

### 예제 입력2 & 예제 출력2

```text
0 2 3

```

```text
1

```

### 예제 입력3 & 예제 출력3

```text
10000000 3 3

```

```text
32768

```

### 예제 입력4 & 예제 출력4

```text
256 2 4

```

```text
89

```

### 예제 입력5 & 예제 출력5

```text
1 1000000 1000000

```

```text
2

```

## 힌트

⌊x⌋는 x를 넘지 않는 가장 큰 정수이다.
